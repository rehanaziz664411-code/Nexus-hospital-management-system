#  Nexus Healthcare Management System

A streamlined Hospital Management application built with **Python**, **Streamlit**, and **SQLite3**. This system allows for patient registration, appointment scheduling, and automated billing records.

##  Features
* **Patient Registration:** Securely store patient details (ID, Age, Blood Group, etc.).
* **Doctor Database:** Pre-configured specialist directory (Cardiology, Neurology, etc.).
* **Appointment Booking:** Link patients to doctors with automatic timestamping.
* **Billing & History:** Searchable patient records with total revenue calculations.
* **Responsive UI:** Custom CSS styling for a professional medical interface.

##  Tech Stack
* **Frontend:** Streamlit
* **Database:** SQLite3
* **Data Handling:** Pandas
* **Styling:** HTML/CSS (Embedded in Streamlit)
